About This App

  1. At first set the number of question
  2. Click the correct answers from your textbook or any other source
  3. Start answering. You can note the number of the questions, which you are confused about, at the box below
  4. You can turn on timer by clicking timer button
  5. Timer will automatically start when you start answering. But for any emergency you can click the stop button while answering and then click the start button again
  6. Reset button will clear all the answers while inserting correct answers or answering. But after getting final result, it will reset everything
